html

A tag html é a tag estrutural principal de um documento HTML. Ela declara para o navegador que o que estiver a seguir faz parte da estrutura de uma página HTML.

head

A tag head é a tag estrutural que marca as informações que são carregadas antes da página em si. Como a tag meta que pode declarar quais tipos de caracteres podem ser aceitos usando o atributo charset, por exemplo: UTF-8... Ou a tag title que declara qual é o título que aparecerá na aba da página.

meta

A tag meta é a tag estrutural filha da tag head que indica as meta-informações da página. Como o tipo de caracteres aceitos, através do atributo charset ou compatibilidade com navegadores mais antigos como Internet Explorer, através do atibuto content.

title

A tag title é a tag estrutural que indica qual é o título da página que aparecerá na aba do navegador ou na janela quando este é minimizado.

body

A tag body é a tag estrutural onde tudo que lá estiver dentro será renderizado e mostrado ao usuário pelo navegador.

headings

O grupo de tags headings são tags que indicam hierarquicamente do maior título até o menor título. Vão de h1 até h6. Ex.:

Título

sub-título

sub-sub-título

sub-sub-sub-título

sub-sub-sub-sub-título
sub-sub-sub-sub-sub-título

p

A tag p é uma tag que indica que o texto dentro dela é um parágrafo.

sup

A tag sup é uma tag usada para colocar o texto de forma "suspendida". Ex.: a = l2

sub

A tag sub é uma tag usada para colocar o texto de forma "rebaixada". Ex.: H2O

blockquote

A tag blockquote é uma tag utilizada para citações, para redirecionar para a fonte da citação, usamos o atributo cite. Ex.:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

small

A tag small é uma tag que deixa o texto pequenino.

strong

A tag strong é uma tag que deixa o texto em negrito.

i

A tag i é uma tag que deixa o texto em itálico.

u

A tag u é uma tag que deixa o texto sublinhado.

mark

A tag mark é uma tag que funciona como um marca-texto.

hr

A tag hr é uma tag que cria uma separação visual dentre um bloco e outro.

font

A tag font é uma tag depreciada do HTML 4, ela funcionava para mudar tipo da fonte, cor e outros atributos.

del

A tag del é uma tag que risca um texto. Ex.: texto riscado

abbr

A tag abbr é uma tag que serve para indicar que o texto é uma abreviação, utilizamos o atribulo title para indicar o significado da abreviação. Ex.: ONU

ul

A tag ul é uma tag que constrói uma lista sem ordem. Ou seja, os itens não são organizados numericamente.

ol

A tag ol é uma tag que constrói uma lista ordenada. Ou seja, os itens são organizados numericamente. 1,2,3,4...

li

A tag li é uma tag filha de tags ul ou ol. Ela indica cada item de uma lista.
